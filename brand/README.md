# Reframe Aging — Brand Mark System

The mark is a **reserve line you live above**. A single line holds steady, then
rises once — late and gently — above its baseline. That rise is *reserve /
capacity*: the headroom you train for. The logo only hints at the idea; the
Capacity Principle article teaches it.

Wordmark: **Cinzel** (weight 600) — an inscriptional Roman capital that reads as
institute, journal, established practice.
Palette: the site's existing greens, so the mark sits natively in the current site.

Design principle: **subtractive.** Ageless, simple, impactful. The wordmarks are
clean; the reserve line is low and soft; the node is optional; the recurring
expression of the idea lives in the divider, not in the logo.

## Files

| File | Use |
|---|---|
| `logo-primary-stacked.svg` | **Official logo.** Homepage hero, report covers, business cards, assessment PDFs, signage, print. Clean — no demand line, no node. Light surfaces. |
| `logo-primary-stacked-onforest.svg` | Official logo for dark forest surfaces. |
| `logo-primary-stacked-node.svg` | Optional variant **with** the node, for moments where you want the signature touch ("you, risen into the reserve"). Light surfaces. |
| `logo-primary-stacked-node-onforest.svg` | Optional node variant, forest surfaces. |
| `logo-horizontal.svg` | Nav, email signature, letterhead, tight horizontal spaces. Light surfaces. |
| `logo-horizontal-onforest.svg` | Horizontal mark for the dark footer. |
| `favicon-tile.svg` | Favicon, social avatar, app icon. Forest tile + cream RA. Square only. |
| `mark-ra-transparent.svg` | RA monogram, transparent background, ink letters. Small square contexts where a tile isn't wanted. |
| `mark-reserve.svg` | Letter-free reserve mark (curve + node). The one place the node is standard — use as a stamp / accent / watermark. |
| `reserve-line-divider.svg` | **The recurring website motif.** Under major page headers, between article sections, before the footer. This is where the reserve idea spreads across the site. |

## Hierarchy

1. **Stacked wordmark leads.** It is the primary, official mark.
2. **Horizontal is its nav/footer adaptation.**
3. **RA monogram is secondary** — favicon, social, small square only.
4. **The divider carries the motif site-wide.** Repeated lightly, never after every section.

## The node is optional

The default marks are node-free. The node lives standard only on `mark-reserve.svg`.
For the wordmark, reach for the `-node` variant when a moment calls for the
signature — otherwise stay clean.

## Taglines (never welded to the logo)

- **Capacity for What Matters** — homepage / brand message
- **Build Reserve. Expand Life.** — workshops / assessment / education
- **Move Well. Live Well. Age Well.** — softer public language, used sparingly

## Do / Don't

- Do keep the reserve line thin and low — it reads as margin, not motion.
- Do give the mark clear space.
- Don't add the demand line back to the official logo. It was removed on purpose.
- Don't force the node onto the wordmark; it's optional.
- Don't recolor dark-green-on-dark; use the on-forest files.
- Don't stretch, skew, or re-typeset the wordmark in another font.
- Don't repeat the divider after every section — it cheapens it.

## Important: outline the font before portable print/PDF use

These SVGs render the wordmark as **live Cinzel text** via a Google Fonts
`@import`. That displays correctly on the web and anywhere with internet. For
print, PDFs, or any offline/portable use, open each file in Figma, Illustrator, or
Inkscape and **convert the text to outlines** (Type → Create Outlines / Path →
Object to Path) so the lettering travels with the file. The reserve line and node
are already pure vector paths and need no conversion.
